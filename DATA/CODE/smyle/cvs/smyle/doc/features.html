SMYLE: Features of current version (Smyle 0.8.3)

  • Open source
  • Personal or commercial use at no charge
  • Convenient usage - seamlessly integrated with Java
  • Lightweight
  • Structured data types
  • Queries
  • Transactions (commit/rollback)
  • Self-optimizing: Indexes are created as necessary
  • Flexible multidimensional indexes
  • Basic schema evolution performed automatically
  • High reliability
    • Journalling storage model
    • Robust, instant crash recovery
    • Backed by large unit test suite
    • Tried in real-world applications
  • Full thread safety
  • Mostly lock-free read access
  • Pure Java (requires JDK 1.3 or higher)
  • Can be embedded in applications
  • Compatible with GJ (extended Java dialect)
  • Based on Java Collection API where possible
  • Automatic conversion of legacy Smyle 0.1/0.2/0.2.5/0.3 beta/0.8.* stores
